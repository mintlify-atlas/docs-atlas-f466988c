AQBot is a cross-platform desktop application that brings powerful AI conversation directly to your machine — with no cloud account required beyond your own provider API keys. It combines a full-featured AI chat client with a local API gateway, letting you talk to any AI model and expose a local OpenAI-compatible endpoint for other tools to use.

## Get started

<Steps>
  <Step title="Download AQBot">
    Go to the [Releases page](https://github.com/AQBot-Desktop/AQBot/releases) and download the installer for your platform (macOS, Windows, or Linux).
  </Step>
  <Step title="Add a provider">
    Open **Settings → Providers**, add your provider (OpenAI, Claude, Gemini, or any OpenAI-compatible API), and enter your API key.
  </Step>
  <Step title="Start a conversation">
    Click **New Chat**, choose your model, and start talking. Switch providers at any time from the model selector.
  </Step>
  <Step title="Explore advanced features">
    Enable Agent mode for autonomous tasks, set up a Knowledge Base for document Q&A, or start the API Gateway to connect other tools.
  </Step>
</Steps>

<Note>
  AQBot stores all data locally — your conversations, API keys, and settings never leave your machine. API keys are encrypted at rest with AES-256.
</Note>
