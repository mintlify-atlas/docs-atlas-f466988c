title: "Get started with AQBot"
description: "Install AQBot, add your first AI provider (OpenAI, Claude, or Gemini), and send your first AI message in five steps on macOS, Windows, or Linux."
---

This guide walks you through installing AQBot, connecting it to an AI provider, and sending your first message. The whole process takes under five minutes.

<Steps>
  <Step title="Download and install AQBot">
    Go to the [AQBot Releases page](https://github.com/AQBot-Desktop/AQBot/releases) and download the installer for your platform:

    - **macOS**: Download the `.dmg` file for Apple Silicon (`arm64`) or Intel (`x86_64`)
    - **Windows 10/11**: Download the `.msi` or `.exe` installer for your architecture
    - **Linux**: Download the `.AppImage`, `.deb`, or `.rpm` package

    Run the installer and follow the on-screen instructions.

    <Note>
      On macOS, you may see a security warning because AQBot is not signed by Apple. See the [installation guide](/installation) for the exact commands to bypass this.
    </Note>
  </Step>

  <Step title="Launch AQBot for the first time">
    Open AQBot from your Applications folder (macOS), Start menu (Windows), or application launcher (Linux). On first launch, AQBot creates its data directories (`~/.aqbot/` and `~/Documents/aqbot/`) automatically.
  </Step>

  <Step title="Add your first AI provider">
    Before you can chat, you need to connect at least one AI provider:

    1. Click **Settings** (gear icon) in the sidebar
    2. Select **Providers**
    3. Click **Add Provider**
    4. Choose your provider (e.g., **OpenAI**)
    5. Enter your API key

    ```
    Provider: OpenAI
    Base URL: https://api.openai.com/v1  (pre-filled)
    API Key:  sk-...your-key-here...
    ```

    6. Click **Save** — AQBot will fetch the available model list automatically

    <Tip>
      You can add multiple API keys for the same provider. AQBot rotates between them automatically to distribute rate limit pressure. See [Configuring providers](/settings/providers) for details.
    </Tip>
  </Step>

  <Step title="Create your first conversation">
    Return to the main chat view:

    1. Click **New Chat** (pencil icon or `+` button)
    2. In the model selector at the top of the chat, choose a model (e.g., `gpt-4o`)
    3. The conversation is ready — type your first message in the input box
  </Step>

  <Step title="Send a message and explore">
    Type a message and press **Enter** (or click the send button). AQBot streams the response token by token in real time.

    A few things to try next:
    - Switch models mid-conversation using the model selector
    - Ask the same question to multiple models simultaneously using the **Multi-model** option
    - Enable **Agent mode** to let the AI read/write files and run commands
    - Upload a document to the **Knowledge Base** and ask questions about it

    <Tip>
      If a response includes code, AQBot embeds a Monaco editor in the code block — you can copy, view a diff, or edit the code directly.
    </Tip>
  </Step>
</Steps>
